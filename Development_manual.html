About Tic Tac Toe

Tic Tac Toe are two players game, X and O, who takes turns making the spaces in a 3x3 grid. Player wins the game if he succeeds in placing three of their marks in a horizontal, vertical or diagonal row.

Source control

We used Github as the source control client. Go to GitHub and there you can download the project as .zip/.tar.gz, or you can fork and clone the project by copying the link and do:
